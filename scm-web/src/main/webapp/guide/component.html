ETC (컴포넌트들의 조합)

  • 속성들을 form-inline class로 inline화 시켜 한줄로 배열합니다.
  • 속성들을 form-list class로 block화 시켜 내려쓰기로 배열합니다.
  • form-group class는 input태그와 오브젝트들(예. 버튼, 라벨등등..)을 가로사이즈에 상관없이 사용
  • input-group class는 input태그와 오브젝트들(예. 버튼, 라벨등등..)을 조합하여 가로사이즈로 full로 채울때 col-*와 함께 사용

테이블안의 Full 가로사이즈 조합

  • 속성들을 form-inline class로 inline화 시켜 한줄로 배열합니다.
  • 속성들을 form-list class로 block화 시켜 내려쓰기로 배열합니다.
  • form-group class는 input태그와 오브젝트들(예. 버튼, 라벨등등..)을 가로사이즈에 상관없이 사용
  • input-group class는 input태그와 오브젝트들(예. 버튼, 라벨등등..)을 조합하여 가로사이즈로 full로 채울때 col-*와 함께 사용
