상세검색 박스 타입 - 컨텐츠 상단(기본)

  • 속성들을 form-inline class로 inline화 시켜 한줄로 배열합니다.
  • 속성들을 form-list class로 block화 시켜 내려쓰기로 배열합니다.

상세검색 박스 타입 - 컨텐츠 상단(적응형)

  • 속성들을 form-inline class로 inline화 시켜 한줄로 배열합니다.
  • 속성들을 form-list class로 block화 시켜 내려쓰기로 배열합니다.
  • 부트스트랩 그리드 기반 col-* class로 적응형 레이아웃을 구성합니다. (참고 - https://getbootstrap.kr/docs/5.0/layout/grid/)
