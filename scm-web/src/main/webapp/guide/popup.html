팝업오픈

Modal Popup 반응형 사이즈별 분류

  • 반응형은 가로사이즈를 고정하면 안되기 때문에 크게 4가지(Basic, Small, Large, Full)로 사이즈를 나눔
  • Basic 사이즈는 Modal Popup(기본)를 사용
  • Small 사이즈는 class="modal"에 bs-modal-sm를 추가하며, class="modal-dialog"에 modal-sm를 추가
  • Large 사이즈는 class="modal"에 bs-modal-lg를 추가하며, class="modal-dialog"에 modal-lg를 추가
  • Full 사이즈는 class="modal"에 bs-modal-lg를 추가하며, class="modal-dialog"에 modal-full를 추가

확인하려면 아래 버튼을 클릭하십시오.

Small 팝업오픈 Large 팝업오픈 Full 팝업오픈

Modal Popup 스크롤

  • 세로 스크롤 팝업은 class="modal-body"에 scrollOptionYclass를 삽입하며, 인라인에 style="height:[size];"사이즈를 지정합니다.

확인하려면 아래 버튼을 클릭하십시오.
