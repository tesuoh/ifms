이미지 네이밍 Guide

  • 아래 종류는 각각의 목적에 맞는 형태로 사용될 수 있도록 예약되어 있다.
  • 네이밍의 순서는 "형태>의미>상태" 순서이다"
  • 동일한 용도의 이미지가 여러 개인 경우에는 뒤에 숫자나 타입으로 표시
  • 롤오버 시에는, 항상 _on, _off 처리
  • 이미지에 있는 텍스트 명으로 표기 함(띄어쓰기 이미지 표현 그대로)
